Get started with the Wirtual API:

## Quickstart

Generate a token on your Wirtual deployment by visiting:

```shell
https://cloud.wirtual.dev/settings/tokens
```

List your workspaces

```shell
# CLI
curl https://cloud.wirtual.dev/api/v2/workspaces?q=owner:me \
-H "Wirtual-Session-Token: <your-token>"
```
