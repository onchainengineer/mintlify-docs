## Quick Install

<Steps>
  <Step title="Install CLI">
    ```bash
    curl -L https://install.wirtual.dev/install.sh | sh
    ```
  </Step>

  <Step title="Authenticate">
    ```bash
    wirtual login
    ```
  </Step>

  <Step title="Create Environment">
    ```bash
    wirtual create my-first-sim --template basic-robot
    ```
  </Step>
</Steps>
