---
title: 'Environment Templates'
description: 'Pre-configured simulation templates for physical AI development and testing'
---

Environment templates are the foundation of wirtual.dev, providing ready-to-use configurations for common physical AI development scenarios. Each template is pre-configured with the necessary simulation tools, dependencies, and settings optimized for specific use cases.

## Available Templates

<CardGroup cols={2}>
  <Card title="Robotic Arm Simulation" icon="robot" color="#2563EB">
    Training environments for robotic arm control, grasping, and manipulation tasks.

    ```bash
    wirtual create arm-training --template robot-arm
    ```
  </Card>

  <Card title="Autonomous Navigation" icon="car" color="#2563EB">
    Environments for developing and testing autonomous navigation systems.

    ```bash
    wirtual create nav-sim --template autonomous-nav
    ```
  </Card>

  <Card title="Multi-Agent Systems" icon="robot-astromech" color="#2563EB">
    Simulate multiple AI agents interacting in a shared environment.

    ```bash
    wirtual create swarm-sim --template multi-agent
    ```
  </Card>

  <Card title="Custom Development" icon="code" color="#2563EB">
    Base template for custom simulation development.

    ```bash
    wirtual create custom-env --template devcontainer
    ```
  </Card>
</CardGroup>
