# tokens

Manage personal access tokens

Aliases:

- token

## Usage

```console
wirtual tokens
```

## Description

```console
Tokens are used to authenticate automated clients to Wirtual.
  - Create a token for automation:

     $ wirtual tokens create

  - List your tokens:

     $ wirtual tokens ls

  - Remove a token by ID:

     $ wirtual tokens rm WuoWs4ZsMX
```
