# notifications

Manage wirtual notifications

Aliases:

- notification

## Usage

```console
wirtual notifications
```

## Description

```console
Administrators can use these commands to change notification settings.
  - Pause wirtual notifications. Administrators can temporarily stop notifiers from
dispatching messages in case of the target outage (for example: unavailable SMTP
server or Webhook not responding).:

     $ wirtual notifications pause

  - Resume wirtual notifications:

     $ wirtual notifications resume
```
